efm8_examples
==============

Firmware examples for the Silicon Labs EFM8 device family using
Keil, sdcc, and Simplicity Studio toolchains.  Example projects
include standard and bootload (using gboot) builds. 

examples
-----------

| Example       | Description                                                  |
| ------------- | ------------------------------------------------------------ |
